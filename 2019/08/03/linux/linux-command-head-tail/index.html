리눅스의 head / tail 명령어

파일의 앞 부분과 뒷 부분을 출력하는 명령어.

head

파일 앞 부분의 10줄을 출력한다. 다양한 옵션을 사용할 수 있다.

tail

파일 뒷 부분의 10줄을 출력한다

-n 옵션을 이용해서 몇 개의 라인을 출력할지 설정할 수 있다

-f 와 -F 옵션을 사용하면 파일이 내용이 변경되는 것까지 추적할 수 있다
