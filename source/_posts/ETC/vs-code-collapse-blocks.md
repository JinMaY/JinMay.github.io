하나의 파일에 코드의 길이가 길때 다른 부분을 참고해가면서 코딩하기 어려울 때가 종종 있다. 모든 코드 블록을 접고 필요한 부분만 펴서 보면 좋을 것 같아서 단축키를 정리해본다.

### 모든 코드 블록 접기

```sh
# macOS
command + K + 0(숫자 0)

# Window
control + K + 0(숫자 0)
```

### 모든 코드 블록 펼치기

```sh
# macOS
command + K + J

# Window
control + K + J
```
