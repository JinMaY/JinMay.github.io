Double Linked List

  • 이중 연결리스트
  • 단방향성을 가진 단순 연결리스트의 단점을 보완
  • 이전 노드(prev)와 다음 노드(next), 그리고 데이터(data)를 가짐
  • head / tail 키워드가 중요!!

head와 tail에 주의하여야 한다!!

head에서 부터 데이터를 읽기 시작하여 특정 데이터 앞에 새로운 노드를 추가
